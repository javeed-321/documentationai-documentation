## Prerequisites

<Callout kind="info">
Before you begin, create a free account at `https://dashboard.example.com` and obtain your store API key. You need Node.js 18 or later installed.
</Callout>

## Create Your Store

<Steps>
  <Step title="Initialize the project" icon="terminal">
    Run the following command to scaffold a new high-volume store.

    ```bash
    npx shopifymax-cli init my-store --products 100000
    ```
  </Step>
  <Step title="Configure checkout speed" icon="settings">
    Edit `shopifymax.config.js` and set `checkoutTarget` to `<3s`.

    ```javascript
    module.exports = {
      checkoutTarget: "<3s",
      maxProducts: 100000,
    };
    ```
  </Step>
</Steps>

## Import Large Product Catalogs

<CodeGroup tabs="CLI,API">
```bash
shopifymax import --file catalog.csv --batch 5000
```
```javascript
import { importProducts } from "shopifymax";

await importProducts({
  source: "catalog.csv",
  batchSize: 5000,
});
```
</CodeGroup>

## Configure Checkout and Variables

Add support for `{CART_TOKEN}` and `{PAYMENT_KEY}` in your checkout flow.

```javascript
const checkout = await shopifymax.checkout.create({
  cartToken: "{CART_TOKEN}",
  paymentKey: "{PAYMENT_KEY}",
});
```

Checkout completes in under three seconds even with over 100000 products.
