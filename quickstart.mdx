## Prerequisites

<Callout kind="info">
Sign up at `https://dashboard.example.com` and create an API key before you start. You need a store plan that supports more than 100000 products.
</Callout>

## Create and Launch Your Store

<Steps>
<Step title="Create your store" icon="plus-circle">
Go to `https://dashboard.example.com/stores` and select Create Store. Name the store and choose your region.
</Step>
<Step title="Install a theme" icon="download">
From the dashboard, browse themes and click Install on a performance-optimized template. The process completes in under two minutes.
</Step>
<Step title="Enable fast checkout" icon="zap">
Toggle Sub-3-Second Checkout in the store settings. ShopifyMax activates edge caching automatically.
</Step>
</Steps>

## Configure Required Variables

Add the following keys in the Environment Variables section of your store dashboard.

<CodeGroup tabs="Dashboard,CLI">
```bash
shopifymax config set CART_TOKEN your_cart_token_here
shopifymax config set PAYMENT_KEY your_payment_key_here
```
```json
{
  "CART_TOKEN": "your_cart_token_here",
  "PAYMENT_KEY": "your_payment_key_here"
}
```
</CodeGroup>

## Verify Your Setup

Open your store URL in an incognito window and add a test product to the cart. Checkout should finish in less than three seconds.
