## Prerequisites

<Callout kind="info">
Obtain an API key from `https://dashboard.example.com`. The platform supports catalogs exceeding `100000` products and checkouts completing in under `3` seconds.
</Callout>

## Client Setup

<Steps>
  <Step title="Install the package" icon="download">
    Run the following command in your project directory.

    ```bash
    npm install @shopifymax/api
    ```
  </Step>
  <Step title="Configure credentials" icon="settings">
    Set your API key as an environment variable.

    ```bash
    export SHOPIFYMAX_API_KEY=your_key_here
    ```
  </Step>
</Steps>

## Minimal Working Example

<CodeGroup tabs="JavaScript,Python">
```javascript
import { ShopifyMaxClient } from '@shopifymax/api';

const client = new ShopifyMaxClient({ apiKey: process.env.SHOPIFYMAX_API_KEY });

const product = await client.products.create({
  title: 'Wireless Headphones',
  price: 99.99,
  inventory: 500,
});

const checkout = await client.checkouts.create({
  cartToken: '{CART_TOKEN}',
  paymentKey: '{PAYMENT_KEY}',
  productId: product.id,
});

console.log('Checkout completed in under 3 seconds:', checkout.id);
```

```python
from shopifymax import ShopifyMaxClient
import os

client = ShopifyMaxClient(api_key=os.environ['SHOPIFYMAX_API_KEY'])

product = client.products.create({
    'title': 'Wireless Headphones',
    'price': 99.99,
    'inventory': 500,
})

checkout = client.checkouts.create({
    'cart_token': '{CART_TOKEN}',
    'payment_key': '{PAYMENT_KEY}',
    'product_id': product['id'],
})

print('Checkout completed in under 3 seconds:', checkout['id'])
```
</CodeGroup>

## Variable Usage

Pass `{CART_TOKEN}` and `{PAYMENT_KEY}` directly in checkout payloads. These tokens enable the sub-3-second checkout path while the platform handles catalogs larger than `100000` products.
