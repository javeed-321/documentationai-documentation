## Prerequisites

<Callout kind="info">
Install Node.js 18 or later. Obtain an API key from the dashboard at https://dashboard.example.com. The platform supports catalogs exceeding 100000 products and checkouts completing in under 3 seconds.
</Callout>

  </Step>
</Steps>

## First API Calls

Use the following example to sync a large catalog and initiate a checkout. The platform substitutes variables such as `CART_TOKEN` and `PAYMENT_KEY` in requests.

<CodeGroup tabs="cURL,JavaScript">
```bash
curl -X POST https://api.example.com/catalog/sync \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "products": [{"id": "p1", "title": "Example Product"}],
    "count": 100000
  }'

curl -X POST https://api.example.com/checkout/initiate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cart_token": "CART_TOKEN",
    "payment_key": "PAYMENT_KEY"
  }'
```
```javascript
const syncCatalog = async () => {
  const res = await fetch('https://api.example.com/catalog/sync', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      products: [{ id: 'p1', title: 'Example Product' }],
      count: 100000
    })
  });
  return res.json();
};

const initiateCheckout = async () => {
  const res = await fetch('https://api.example.com/checkout/initiate', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      cart_token: 'CART_TOKEN',
      payment_key: 'PAYMENT_KEY'
    })
  });
  return res.json();
};
```
