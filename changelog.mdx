<Update label="2024-11-05" description="v2.1.0" tags={["feature"]}>

## Checkout and variables

- Reduced average checkout time to under 3 seconds through optimized payment routing.
- Added native support for `CART_TOKEN` and `PAYMENT_KEY` variables in checkout flows.

</Update>

<Update label="2024-10-12" description="v2.0.0" tags={["feature"]}>

## Catalog scalability

- Extended product catalog support to handle more than 100000 items with improved indexing.
- Introduced parallel loading for large inventories during search and browse operations.

</Update>

<Update label="2024-09-28" description="v1.9.5" tags={["bugfix"]}>

## Performance fixes

- Resolved timeout errors on high-volume catalog queries exceeding 100000 products.
- Fixed variable substitution failures that affected custom checkout tokens.

</Update>
